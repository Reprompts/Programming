# Array Traversal Mastery Roadmap

---

## LEVEL 1 — Basic Linear Traversal (Foundation)
**Goal:** Become completely comfortable with `0 → n-1` traversal.

### Problems:
- Print all elements of an array  
- Find the sum of all elements in an array  
- Find the maximum and minimum element in an array  
- Count how many even and odd numbers exist in the array  
- Print all elements greater than a given number `K`  

---

## LEVEL 2 — Conditional Traversal
**Goal:** Learn skipping elements and applying conditional logic.

### Problems:
- Print all negative numbers in an array  
- Count elements divisible by 3 but not by 5  
- Print indices of all occurrences of a target element  
- Find the first element greater than `K`  
- Print all elements except duplicates *(skip repeating adjacent elements in sorted array)*  

### Skills Learned:
- `continue`
- `break`
- Conditional traversal
- Filtering logic  

---

## LEVEL 3 — Reverse Traversal
**Goal:** Become comfortable with `n-1 → 0` iteration.

### Problems:
- Print the array in reverse order  
- Find the last occurrence of an element  
- Check if array is palindrome using reverse traversal  
- Print elements from the last even index to the beginning  
- Reverse an array in place using a single loop  

### Important Concepts:
- Reverse indexing  
- Boundary errors  

---

## LEVEL 4 — Adjacent Element Traversal
**Goal:** Work with neighboring elements safely.

### Problems:
- Count number of adjacent equal pairs  
- Find first increasing pair (`arr[i] < arr[i+1]`)  
- Count number of peaks  

Peak Condition:
arr[i] > arr[i-1] AND arr[i] > arr[i+1]


- Replace every element with difference from next element  
- Find the largest difference between adjacent elements  

### Skills Gained:
- Safe use of `i+1`  
- Safe use of `i-1`  
- Loop boundary design  

---

## LEVEL 5 — Two Pointer Traversal
**Goal:** Learn multi-pointer movement.

### Problems:
- Reverse an array using two pointers  
- Check if array is palindrome using two pointers  
- Move all zeros to end of array  
- Partition array into negatives and positives  
- Remove duplicates from sorted array using two pointers  

### Skills Gained:
- Pointer movement  
- In-place modification  
- Memory efficiency  

---

## LEVEL 6 — Multi Pointer + Conditional Traversal
**Goal:** Combine traversal logic with pointer control.

### Problems:
- Find two numbers that sum to target *(sorted array)*  
- Count pairs with sum less than target  
- Merge two sorted arrays using traversal  
- Find intersection of two sorted arrays  
- Find missing number in sorted array  

### Skills Learned:
- Simultaneous traversal  
- Comparison-based movement  

---

## LEVEL 7 — Nested Traversal (2D Arrays)
**Goal:** Understand row-wise and column-wise traversal.

### Problems:
- Print a 2D matrix row-wise  
- Print matrix column-wise  
- Find sum of each row  
- Find sum of each column  
- Print main diagonal and secondary diagonal  

### Skills Gained:
- Nested loops  
- 2D indexing  

---

## LEVEL 8 — Advanced Traversal Patterns
**Goal:** Master complex traversal patterns.

### Problems:
- Print matrix in spiral order  
- Traverse matrix in zig-zag order  
- Find largest element in each row and columns  
- Rotate matrix 90 degrees clockwise  
- Traverse matrix diagonal by diagonal  

### Skills Gained:
- Boundary control  
- Complex iteration logic  

---
