## Question 58
**Problem Statement:**  
The chef has to travel to another place. For this, he can avail any one of two cab services.  
- The first cab service charges X rupees.  
- The second cab service charges Y rupees.  

The chef wants to spend the minimum amount of money.  

**Input Format:**  
- The first and only line contains two integers X and Y.  

**Output Format:**  
- Output `FIRST` if the first cab is cheaper  
- Output `SECOND` if the second cab is cheaper  
- Output `ANY` if both cost the same  

**Constraints:**  
- 1 ≤ T ≤ 100  

## Question 59
**Problem Statement:**  
The chef is watching TV. The current volume is X.  
- Volume up increases by 1  
- Volume down decreases by 1  

Find minimum button presses to change volume from X to Y.  

**Input Format:**  
- First line: integer T (test cases)  
- Next lines: two integers X and Y  

**Output Format:**  
- Minimum number of button presses  

**Constraints:**  
- 1 ≤ T ≤ 100  
- 1 ≤ X ≤ Y ≤ 100
